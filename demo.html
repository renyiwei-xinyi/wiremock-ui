📊

仪表板监控

  • 映射数量统计
  • 请求匹配率分析
  • 快速操作入口

  • JSON 代码编辑器

🚀 快速开始

1

安装 Node.js

从官网下载并安装 Node.js 18+ 版本

node --version
2

启动 WireMock

启动 WireMock 服务在 8080 端口

java -jar wiremock-standalone.jar --port 8080
3

安装依赖

在项目目录下安装所需依赖

npm install
4

启动应用

启动开发服务器并访问应用

npm run dev
